# FUTURE_ML_03
Resume Screening and Candidate Ranking System using NLP, TF-IDF Vectorization, and Cosine Similarity for automated recruitment support.
Resume Screening & Candidate Ranking System Using NLP
Recruiters often receive hundreds of resumes for a single job opening. Manually reviewing and ranking candidates is time-consuming and inefficient.

To address this challenge, I built an NLP-powered Resume Screening System that automatically evaluates resumes against a job description and ranks candidates based on relevance.

🔹 Key Features

✅ Resume Text Cleaning & Preprocessing

✅ NLP-Based Skill Extraction

✅ Job Description Analysis

✅ TF-IDF Vectorization

✅ Cosine Similarity Scoring

✅ Candidate Ranking System

✅ Skill Gap Identification

🛠️ Technologies Used

• Python

• Pandas

• NLTK

• Scikit-Learn

• TF-IDF Vectorization

• Cosine Similarity

• Matplotlib

• Jupyter Notebook

📊 Project Outcome

The system compares resumes with a target job role, calculates similarity scores, ranks candidates, and identifies missing skills that may impact job fit.

This solution can help recruiters:

• Shortlist candidates faster

• Improve hiring efficiency

• Reduce manual screening effort
